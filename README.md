# 산청군 3D 디지털 트윈 수해 취약지역 시각화

경상남도 산청군 2025년 수해 취약지역을 3D로 시각화한 디지털 트윈 프로젝트입니다.

## 주요 기능

- 🏔️ **3D 지형 모델**: 산청군의 실제 지형을 glTF 포맷으로 표현
- 🌊 **침수 시뮬레이션**: 강수량에 따른 침수 수위 시각화
- ☀️ **시간대별 조명**: 하루 24시간 동안의 태양 위치 변화
- 📊 **강수량 예측**: 침수 수위별 예상 강수량 표시
- 🎮 **인터랙티브 컨트롤**: 마우스로 회전, 확대/축소, 이동

## 기술 스택

- **Three.js**: 3D 렌더링
- **Vite**: 빌드 도구
- **glTF/GLB**: 3D 모델 포맷

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## 사용 방법

1. **카메라 조작**
   - 마우스 좌클릭 드래그: 회전
   - 마우스 휠: 확대/축소
   - 마우스 우클릭 드래그: 이동

2. **시간 조절**
   - 우측 상단 시간 슬라이더로 하루 중 시간대 조절
   - 새벽, 낮, 저녁, 밤의 조명 변화 확인

3. **침수 시뮬레이션**
   - "침수지역 표시" 버튼 클릭
   - 우측 하단 슬라이더로 침수 수위 조절 (0-100%)
   - 강수량에 따른 위험도 확인

## 침수 수위 기준

- **0-30%**: 안전 (0-100mm 강수)
- **30-50%**: 주의 (100-200mm 강수)
- **50-70%**: 경고 (200-350mm 집중호우)
- **70-100%**: 위험 (350-500mm 극한 강수)
